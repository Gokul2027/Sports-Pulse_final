# ⚽ Sports Pulse

A web platform for college students to explore, register, and participate in sports events, along with an alumni interaction section.

## 🚀 Features

- 🏆 View upcoming sports events
- 📝 Register for events online
- 🎓 Alumni connect section (experience sharing)
- 📊 Event tracking and participation details
